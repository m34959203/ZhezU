# ZhezU — Новый сайт Жезказганского университета им. О.А. Байконурова

> Проект разработки нового сайта для **Жезказганского университета имени О.А. Байконурова** (https://zhezu.edu.kz/).
> Цель — создание современной, мультиязычной цифровой платформы, которая заменит текущий сайт университета.

---

## О проекте

**ZhezU** — это полноценное веб-приложение нового поколения, разрабатываемое для обновления официального сайта университета по адресу **https://zhezu.edu.kz/**. Проект объединяет приёмную комиссию, академический каталог, карьерный центр, студенческие сервисы и административные инструменты в единой платформе.

### Целевая аудитория

- **Абитуриенты и родители** — информация о приёме, образовательных программах, подача заявок
- **Текущие студенты** — личный кабинет, талант-профиль, карьерные сервисы
- **Преподаватели и кураторы** — панель управления, коммуникация со студентами
- **Работодатели-партнёры** — пул талантов, профили студентов
- **Администрация университета** — CMS, аналитика, управление контентом

### Ключевые возможности

- Мультиязычность: **қазақша**, **русский**, **English**
- Тёмная и светлая темы
- Адаптивный дизайн (мобильные устройства, планшеты, десктоп)
- PWA (Progressive Web App) — установка на мобильные устройства
- Оптимизация SEO для поисковых систем
- Доступность (WCAG 2.1 AA)

## Быстрый старт

### Требования

- Node.js 22+
- pnpm 9+

### Установка и запуск

```bash
# Клонирование репозитория
git clone <repository-url>
cd ZhezU/zhezu-app

# Установка зависимостей
pnpm install

# Запуск в режиме разработки
pnpm dev

# Сборка для продакшена
pnpm build

# Запуск продакшен-сервера
pnpm start
```

Приложение будет доступно по адресу `http://localhost:3000`.

### Docker

```bash
# Сборка образа
docker build -t zhezu-app .

# Запуск контейнера
docker run -p 3000:3000 zhezu-app
```
