## UI-КОМПОНЕНТЫ

### Button
- Варианты: primary (синий фон), secondary (золотой фон), outline (обводка), ghost (прозрачный)
- Размеры: sm, md, lg
- Состояния: default, hover (яркость +10%), active, disabled (opacity 0.5), loading (спиннер)

### Card
- Варианты: default (белый фон, тень), elevated (больше тени), bordered (обводка), interactive (hover lift)
- Скругление: rounded-xl (12px)
- Padding: 24px

### Badge
- Варианты: default (серый), primary (синий), success (зелёный), warning (жёлтый), error (красный), info (голубой)
- Размер: компактный, округлый

### Input
- Поле ввода с label сверху, placeholder, иконка слева (опционально)
- Состояния: default, focus (синяя обводка), error (красная обводка + текст ошибки), disabled

### Tabs
- Горизонтальные вкладки с подчёркиванием активной (синяя линия снизу)

### Accordion
- Заголовок + иконка ChevronDown, при раскрытии — плавная анимация содержимого

### Skeleton
- Серые пульсирующие блоки для загрузки контента

### Toast/Notification
- 4 варианта: success, error, warning, info
- Появляется справа сверху с анимацией slide-in

### Search Overlay (Command Palette)
- При нажатии на иконку поиска — полноэкранный оверлей с полем поиска
- Клавиша Cmd+K / Ctrl+K
- Группированные результаты: Программы, Страницы, Новости
- Навигация стрелками клавиатуры

- Скругления: rounded-lg (8px) для кнопок, rounded-xl (12px) для карточек
- Тени: мягкие (shadow-sm для кнопок, shadow-md для карточек, shadow-xl для dropdown)
