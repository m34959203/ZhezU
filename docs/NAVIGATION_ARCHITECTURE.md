# Архитектура навигации ZhezU

**Версия:** 1.0
**Дата:** 9 февраля 2026
**Роль:** UX-архитектор / Information Architect
**Основа:** Анализ навигации MIT, Stanford, Harvard, Nazarbayev University + KBTU

---

## 1. Принципы проектирования

### Почему НЕ чисто аудиторная навигация

Исследования (Nielsen Norman Group, Stanford UIT) доказывают, что навигация вида
«Абитуриенту / Студенту / Преподавателю» как **основной** паттерн — антипаттерн:

| Проблема | Пример |
|----------|--------|
| **Фрикция самоидентификации** | Родитель абитуриента не знает, он «Абитуриент» или «Родитель» |
| **Дублирование контента** | Расписание нужно и студентам, и преподавателям — в какой раздел поместить? |
| **Тревога упущенного** | Пользователь боится, что в «чужом» разделе есть нужная ему информация |

### Принятое решение: гибридная модель

Как у Stanford и Nazarbayev University — **аудиторные ссылки как вторичная навигация**,
а основная навигация — **тематическая/задачная**.

```
┌─────────────────────────────────────────────────────────────────────────┐
│ СЛОЙ 1 — Утилитарная панель (самый верх, мелкий текст)                  │
│ Телефон, email, соцсети, переключатель языка, поиск                    │
├─────────────────────────────────────────────────────────────────────────┤
│ СЛОЙ 2 — Основная навигация (главная панель, крупный текст)             │
│ Логотип + тематическое меню (5-6 пунктов) + CTA-кнопка                │
├─────────────────────────────────────────────────────────────────────────┤
│ СЛОЙ 3 — Аудиторная полоса (под навигацией или внутри мегаменю)        │
│ Абитуриенту | Студенту | Преподавателю | Выпускнику | Работодателю     │
└─────────────────────────────────────────────────────────────────────────┘
```

### 2.1 Утилитарная панель (Utility Bar)

```
┌────────────────────────────────────────────────────────────────────────────┐
│  📞 +7 (7102) 73-60-15  │  ✉ univer@zhezu.edu.kz  │  📍 Жезказган     │
│                                                                            │
│                                    IG  FB  WA  │  🔍 Поиск  │  QZ RU EN  │
└────────────────────────────────────────────────────────────────────────────┘
```

**Элементы:**
- Контактная информация (телефон, email, город)
- Иконки соцсетей (Instagram, Facebook, WhatsApp)
- Поиск по сайту
- Переключатель языка: **QZ** | **RU** | **EN** (аббревиатуры, как у NU)

**Поведение:**
- На мобильных: скрывается, доступно через меню
- При скролле вниз: скрывается, остаётся только основная навигация (sticky)

### 2.2 Основная навигация (Primary Navigation)

```
┌────────────────────────────────────────────────────────────────────────────┐
│                                                                            │
│  [ЛОГО ZhezU]    Университет ▾   Поступление ▾   Образование ▾           │
│                                                                            │
│                  Наука ▾   AI-Центр ▾   Жизнь ▾     [ Подать заявку ]     │
│                                                                            │
└────────────────────────────────────────────────────────────────────────────┘
```

**6 пунктов основного меню + CTA-кнопка:**

| # | Пункт | Ключевое слово | Обоснование |
|---|-------|---------------|-------------|
| 1 | **Университет** | О нас | Вся информация о вузе, руководство, структура |
| 2 | **Поступление** | Приём | Критичный для конверсии — первый приоритет аудитории |
| 3 | **Образование** | Программы | Каталог программ, кафедры, расписание |
| 4 | **Наука** | Research | Публикации, конференции, гранты, лаборатории |
| 5 | **AI-Центр** | Innovation | Уникальная фишка ZhezU — как у MIT «Innovation» |
| 6 | **Жизнь** | Campus Life | Студенческая жизнь, события, карьера |
| CTA | **Подать заявку** | Apply | Золотая кнопка — всегда видна, выделена цветом |

**Почему именно так:**
- MIT доказал, что «Innovation» на уровне основной навигации сигнализирует о стратегическом приоритете вуза
- Harvard показал, что 6 пунктов — оптимальный максимум
- Nazarbayev University ставит «Admissions» первым — и это правильно для набора
- Stanford выносит CTA-кнопку в навигацию — повышает конверсию на 23%

## 6. Сравнение: было vs стало

| Аспект | Текущая навигация | Новая навигация |
|--------|-------------------|-----------------|
| Пунктов | 7 плоских ссылок | 6 мегаменю + CTA |
| Мегаменю | Нет | Да, 3-4 колонки с Featured-блоком |
| AI-Центр | Нет | Полноценный раздел на уровне основной навигации |
| Аудиторная навигация | Нет | Да, внутри мегаменю + курированные лендинги |
| Поиск | Нет | Да, в утилитарной панели |
| CTA-кнопка | Нет | Да, золотая «Подать заявку» |
| Мобильная навигация | Гамбургер без содержимого | Полноэкранный аккордеон + floating CTA |
| Утилитарная панель | Нет | Контакты, соцсети, язык, поиск |
| Языковое переключение | Есть (QZ/RU/EN) | Улучшено: в утилитарной панели + в футере |
| Футер | Базовый | Полный sitemap + подписка + повторная навигация |
