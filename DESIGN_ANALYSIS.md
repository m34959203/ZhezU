## 2. Критическая проблема безопасности

> **ВНИМАНИЕ:** На момент анализа (февраль 2026) основной домен `zhezu.edu.kz` скомпрометирован. Вместо университетского сайта отображается мошеннический контент (онлайн-гемблинг), замаскированный под e-commerce платформу.

### Признаки компрометации:
- Подмена контента на страницы "SLOT DANA" (индонезийский онлайн-гемблинг)
- Использование шаблонов Lazada для маскировки
- Встроенные Facebook Pixel и Google Analytics трекеры злоумышленников
- Кнопки "LOGIN/DAFTAR" перенаправляют на внешние сайты казино
- SEO-спам с фальшивыми товарными карточками

### Рекомендации:
1. **Немедленно** обратиться к хостинг-провайдеру и CERT-KZ
2. Провести полный аудит безопасности серверной инфраструктуры
3. Восстановить сайт из резервной копии
4. Обновить все CMS, плагины и серверное ПО
5. Внедрить WAF (Web Application Firewall)
6. Настроить мониторинг целостности файлов

### 3.1 Цветовая схема

| Роль              | Цвет           | HEX       | Применение                        |
|-------------------|----------------|-----------|-----------------------------------|
| Основной          | Фиолетовый     | `#8E44AD` | Брендинг, заголовки, навигация    |
| Текст основной    | Тёмно-серый    | `#334155` | Основной текст контента           |
| Текст заголовков  | Графитовый     | `#1E293B` | Заголовки разделов                |

| Фон секций        | Белый           | `#FFFFFF` | Карточки, блоки контента          |
| Акцент            | Светло-серый    | `#ADB6BE` | Второстепенные элементы, разделители |

**Оценка цветовой схемы:**
- (+) Фиолетовый — уникальный выбор, выделяет среди конкурентов
- (-) Ограниченная палитра — нет явного акцентного цвета для CTA-кнопок
- (-) Недостаточный контраст некоторых текстовых элементов
- (-) Отсутствие поддержки тёмной темы

## 4. Оценка UI/UX

### 4.1 Матрица оценки

| Критерий                        | Оценка (1–10) | Комментарий                                                    |
|---------------------------------|:-------------:|----------------------------------------------------------------|
| Визуальная привлекательность    | 4/10          | Устаревший дизайн, не соответствует трендам 2025–2026          |
| Навигация и информационная архитектура | 3/10   | Слишком простая, не учитывает разнообразие аудиторий           |
| Адаптивность (responsive)       | 5/10          | Базовые брейкпоинты есть, но реализация минимальная            |
| Типографика и читаемость        | 6/10          | Хорошие шрифты, но нет чёткой иерархии                        |
| Цветовая система                | 5/10          | Фиолетовый — уникальный выбор, но палитра ограничена          |
| Контент и UX-копирайтинг       | 3/10          | Контент не адаптирован под потребности целевых аудиторий       |
| Доступность (a11y)              | 3/10          | Минимальные средства доступности                                |
| Скорость загрузки               | 4/10          | Нет видимых оптимизаций                                        |
| SEO                             | 2/10          | Критический уровень — сайт скомпрометирован                   |
| Безопасность                    | 1/10          | Сайт взломан                                                   |
| **Общая оценка**                | **3.6/10**    | **Требуется полный редизайн**                                  |

### 4.2 Ключевые проблемы

1. **Безопасность (критическая)** — сайт скомпрометирован злоумышленниками
2. **Устаревший дизайн** — не соответствует современным стандартам веб-дизайна университетов
3. **Слабая навигация** — нет мегаменю, поиска, быстрых путей для аудиторий
4. **Отсутствие персонализации** — нет разделения контента по типам пользователей
5. **Нет виртуального тура** — критически важно для привлечения абитуриентов
6. **Отсутствие интерактивных элементов** — чат-бот, фильтры программ, калькуляторы
7. **Нет тёмной темы** — стандартный запрос аудитории Gen Z

## 6. Рекомендации по редизайну в Google Stitch

### 6.1 Возможности Google Stitch для проекта

[Google Stitch](https://stitch.withgoogle.com/) — AI-инструмент от Google Labs, который позволяет:

- **Text-to-UI:** генерировать полные макеты интерфейсов по текстовому описанию

- **Варианты дизайна:** генерация нескольких вариантов для сравнения
- **Экспорт в Figma:** одним кликом скопировать дизайн в Figma с редактируемыми слоями
- **Экспорт кода:** получить чистый HTML/CSS или React-код
- **Live-аудит:** проверка контрастности, отступов, консистентности токенов
